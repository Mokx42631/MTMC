# ⚔️ MCMT — Mokx's Character Management Tool

A comprehensive **D&D 5e (2014)** character sheet manager for Android.

## 📥 Download

**[Download APK](https://mokx42631.github.io/MTMC/mcmt-latest.apk)** · [Visit Site](https://mokx42631.github.io/MTMC/)

## ✨ Features

- **546 spells** · **145 subclasses** · **13 classes** · **472 races** · **141 backgrounds** · **188 feats**
- Multiclass support with level-up feature tracking
- Ritual casting system
- Spell slot tracking with cast/restore buttons
- PDF character sheet import
- Portrait upload & class artwork
- Death save tracking & temp HP
- Currency converter
- Dark mode toggle
- Works fully offline — no internet required
- All data saved locally on your device

## 📱 Installation

1. Download the APK from the link above
2. On your Android device, enable **Install from unknown sources** if prompted
3. Open the APK and install
4. That's it — no account needed, no internet required

## 🛠️ Tech

- HTML / CSS / JavaScript wrapped in Android WebView
- localStorage for persistence
- Fully self-contained — all game data bundled offline
